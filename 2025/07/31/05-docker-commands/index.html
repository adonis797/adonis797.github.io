Docker 常用命令详解

Docker作为当今最流行的容器化平台，掌握其常用命令对于开发、测试和部署环境至关重要。本文将详细介绍Docker的基础概念和常用命令，帮助你高效使用Docker。

1. Docker 基础命令

命令 功能 常用操作
docker version 查看版本 - docker version：显示Docker客户端和服务器版本
  • docker -v：简洁版本信息 |
    | docker info | 系统信息 | - docker info：显示容器数、镜像数、存储驱动等系统信息 |
    | docker help | 帮助文档 | - docker help：显示Docker命令列表
  • docker command --help：特定命令的帮助信息 |

2. 镜像管理命令

命令 功能 常用操作
docker images 列出镜像 - docker images：列出本地所有镜像
  • docker images -a：包括中间层镜像
  • docker images -q：仅显示镜像ID |
    | docker pull | 拉取镜像 | - docker pull ubuntu:20.04：拉取指定版本镜像
  • docker pull nginx：不指定版本则拉取最新版 |
    | docker rmi | 删除镜像 | - docker rmi nginx：删除指定镜像
  • docker rmi $(docker images -q)：删除所有镜像 |
    | docker build | 构建镜像 | - docker build -t myapp:1.0 .：从当前目录的Dockerfile构建镜像
  • docker build -f custom.dockerfile -t myapp .：使用自定义Dockerfile |
    | docker tag | 标记镜像 | - docker tag nginx:latest mynginx:v1：为镜像添加新标签 |
    | docker save | 导出镜像 | - docker save -o nginx.tar nginx:latest：将镜像保存为文件 |
    | docker load | 导入镜像 | - docker load -i nginx.tar：从文件加载镜像 |
    | docker history | 查看历史 | - docker history nginx：查看镜像构建历史 |

3. 容器管理命令

命令 功能 常用操作
docker run 运行容器 - docker run -d nginx：后台运行容器
  • docker run -it ubuntu bash：交互式运行
  • docker run -p 8080:80 nginx：端口映射
  • docker run -v /host/path:/container/path nginx：挂载卷
  • docker run --name mycontainer nginx：指定容器名称 |
    | docker ps | 列出容器 | - docker ps：列出运行中的容器
  • docker ps -a：列出所有容器（包括停止的）
  • docker ps -q：仅显示容器ID |
    | docker stop | 停止容器 | - docker stop container_id：优雅停止容器
  • docker stop $(docker ps -q)：停止所有运行中的容器 |
    | docker start | 启动容器 | - docker start container_id：启动已停止的容器 |
    | docker restart | 重启容器 | - docker restart container_id：重启容器 |
    | docker kill | 强制停止 | - docker kill container_id：强制停止容器 |
    | docker rm | 删除容器 | - docker rm container_id：删除已停止的容器
  • docker rm -f container_id：强制删除运行中的容器
  • docker rm $(docker ps -aq)：删除所有容器 |
    | docker exec | 执行命令 | - docker exec -it container_id bash：进入容器交互式终端
  • docker exec container_id ls /app：在容器中执行命令 |
    | docker logs | 查看日志 | - docker logs container_id：查看容器日志
  • docker logs -f container_id：实时跟踪日志
  • docker logs --tail 100 container_id：查看最后100行 |

4. 网络管理命令

命令 功能 常用操作
docker network ls 列出网络 - docker network ls：列出所有网络
docker network create 创建网络 - docker network create mynetwork：创建网络
  • docker network create --driver bridge mynetwork：指定驱动类型 |
    | docker network connect | 连接网络 | - docker network connect mynetwork container_id：将容器连接到网络 |
    | docker network disconnect | 断开网络 | - docker network disconnect mynetwork container_id：将容器从网络断开 |
    | docker network inspect | 检查网络 | - docker network inspect mynetwork：查看网络详细信息 |
    | docker network rm | 删除网络 | - docker network rm mynetwork：删除网络 |

5. 数据卷管理命令

命令 功能 常用操作
docker volume ls 列出卷 - docker volume ls：列出所有数据卷
docker volume create 创建卷 - docker volume create myvolume：创建数据卷
docker volume inspect 检查卷 - docker volume inspect myvolume：查看卷详细信息
docker volume rm 删除卷 - docker volume rm myvolume：删除数据卷
  • docker volume prune：删除所有未使用的数据卷 |
