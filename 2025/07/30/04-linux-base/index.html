Linux 基础命令详解

Linux作为一个强大的开源操作系统，掌握其命令行操作是每个开发者和系统管理员的必备技能。本文将系统地介绍Linux常用命令，帮助你高效地使用Linux系统。

1. 文件编辑与查看

命令 功能 常用操作
vi/vim 文本编辑器 - vi 文件名：打开文件
  • 编辑模式：按 i 进入
  • 命令模式：按 ESC 退出编辑
  • :wq 保存退出
  • :q! 不保存强制退出
  • /关键词 搜索内容
  • dd 删除当前行
  • yy 复制当前行
  • p 粘贴
  • u 撤销操作 |
    | nano | 简易编辑器 | - nano 文件名：打开文件
  • Ctrl+O：保存
  • Ctrl+X：退出
  • Ctrl+K：剪切行
  • Ctrl+U：粘贴 |
    | cat | 查看文件内容 | - cat 文件名：显示全部内容
  • cat -n 文件：显示行号
  • cat file1 file2 > 合并文件：合并文件
  • cat > 文件名：从键盘创建新文件 |
    | more/less | 分页查看 | - more 大文件：空格翻页，q退出
  • less 大文件：支持上下键滚动（推荐）
  • less -N 文件：显示行号
  • /关键词：搜索 |
    | head/tail | 查看文件头/尾 | - head -n 20 文件：查看前20行
  • tail -n 30 文件：查看后30行
  • tail -f 日志文件：实时查看日志更新 |

2. 文件与目录操作

命令 功能 常用操作
ls 列出目录内容 - ls：列出文件和目录
  • ls -l：详细信息（权限、所有者等）
  • ls -a：显示隐藏文件
  • ls -lh：以人类可读方式显示大小
  • ls -R：递归显示子目录 |
    | cd | 切换目录 | - cd /path/to/dir：切换到指定目录
  • cd ..：返回上级目录
  • cd ~：返回用户主目录
  • cd -：返回上次访问的目录 |
    | pwd | 显示当前路径 | - pwd：显示当前完整工作目录 |
    | mkdir | 创建目录 | - mkdir 目录名：创建目录
  • mkdir -p a/b/c：创建多级目录 |
    | rmdir | 删除空目录 | - rmdir 目录名：删除空目录 |
    | rm | 删除文件/目录 | - rm 文件名：删除文件
  • rm -r 目录：递归删除目录及内容
  • rm -f 文件：强制删除
  • rm -rf 目录：强制递归删除（危险命令） |
    | cp | 复制文件/目录 | - cp 源文件 目标文件：复制文件
  • cp -r 源目录 目标目录：递归复制目录
  • cp -p 源文件 目标文件：保留权限和时间戳 |
    | mv | 移动/重命名 | - mv 源文件 目标文件：移动或重命名文件
  • mv 源目录 目标目录：移动目录
  • mv file1 file2 dir/：移动多个文件到目录 |
    | touch | 创建空文件 | - touch 文件名：创建空文件或更新时间戳 |
    | find | 查找文件 | - find /path -name "*.txt"：按名称查找
  • find /path -type f -size +10M：查找大于10M的文件
  • find /path -mtime -7：查找7天内修改的文件
  • find /path -name "*.log" -exec rm {} \;：查找并执行操作 |

3. 文本处理三剑客

命令 功能 示例
grep 文本搜索 - grep "error" log.txt：查找包含error的行
  • grep -i "warning"：忽略大小写
  • grep -r "func" ./src：递归搜索目录
  • grep -v "info"：反向匹配（不包含info的行）
  • grep -n "keyword"：显示行号
  • grep -A 3 -B 2 "error"：显示匹配行及其前2行后3行 |
    | awk | 文本分析处理 | - awk '{print $1}' file：打印第一列
  • awk -F: '{print $1}' /etc/passwd：以冒号分隔
  • awk '/error/ {count++} END {print count}'：统计出现次数
  • awk '{sum+=$1} END {print sum}'：求和
  • awk 'length($0) > 80'：打印长度超过80的行 |
    | sed | 流编辑器 | - sed 's/old/new/g' file：替换文本
  • sed '5d' file：删除第5行
  • sed -n '10,20p' file：打印10-20行
  • sed '1,5s/old/new/g' file：只替换1-5行
  • sed -i 's/old/new/g' file：直接修改文件
  • sed '/^$/d' file：删除空行 |
    | sort | 排序 | - sort file：按字母顺序排序
  • sort -n file：按数字排序
  • sort -r file：逆序排序
  • sort -k2 file：按第二列排序
  • sort -u file：排序并去重 |
    | uniq | 去重 | - uniq file：去除相邻的重复行
  • sort file | uniq：完全去重
  • uniq -c file：统计重复次数
  • uniq -d file：只显示重复行 |
    | cut | 剪切文本 | - cut -d: -f1 /etc/passwd：以冒号分隔并提取第一列
  • cut -c1-5 file：提取每行前5个字符 |
    | tr | 替换字符 | - tr 'a-z' 'A-Z' < file：小写转大写
  • tr -d '\r' < file：删除回车符
  • tr -s ' '：压缩连续空格为一个 |
    | wc | 统计 | - wc -l file：统计行数
  • wc -w file：统计单词数
  • wc -c file：统计字节数 |

4. 管道与重定向

操作符 功能 示例
**` `** 管道
  • cat file | sort | uniq：排序并去重
  • ps aux | grep nginx：查找nginx进程 |
    | > | 输出重定向 | - ls > files.txt：将输出写入文件（覆盖）
  • echo "text" > file.txt：写入文本到文件 |
    | >> | 追加重定向 | - echo "new line" >> file.txt：追加内容到文件末尾 |
    | < | 输入重定向 | - sort < unsorted.txt：从文件读取输入
  • tr 'a-z' 'A-Z' < file.txt：转换文件内容为大写 |
    | 2> | 错误重定向 | - find / -name core 2> errors.txt：将错误信息写入文件 |
    | &> | 全部重定向 | - ls -la &> output.txt：将标准输出和错误都写入文件 |
    | /dev/null | 空设备 | - command > /dev/null 2>&1：丢弃所有输出 |

5. 网络诊断工具

命令 功能 示例
ping 测试网络连通性 - ping baidu.com：测试连接
  • ping -c 4 8.8.8.8：发送4个包
  • ping -i 0.5 host：每0.5秒发送一次
  • ping -s 1500 host：指定包大小 |
    | telnet | 测试端口连通性 | - telnet 192.168.1.1 80：检查80端口
  • telnet smtp.163.com 25：测试邮件服务
  • telnet localhost 3306：测试MySQL连接 |
    | netstat | 网络状态查看 | - netstat -tuln：查看监听端口
  • netstat -anp | grep :22：检查SSH连接
  • netstat -s：显示网络统计
  • netstat -r：显示路由表
  • netstat -ie：显示网络接口 |
    | ifconfig/ip | 网络接口配置 | - ifconfig：显示所有网络接口
  • ifconfig eth0：显示特定接口
  • ip addr show：显示IP地址
  • ip route show：显示路由信息 |
    | nslookup/dig | DNS查询 | - nslookup baidu.com：查询域名解析
  • dig baidu.com：详细DNS信息
  • dig +short baidu.com：仅显示IP地址 |
    | traceroute | 路由跟踪 | - traceroute baidu.com：跟踪数据包路由
  • traceroute -n google.com：不解析主机名 |
    | curl/wget | 文件下载 | - curl -O https://example.com/file.zip：下载文件
  • wget https://example.com/file.zip：下载文件
  • curl -I https://baidu.com：只获取HTTP头
  • curl -d "param=value" URL：发送POST请求 |
    | ssh | 远程登录 | - ssh user@host：连接到远程主机
  • ssh -p 2222 user@host：指定端口
  • ssh -i key.pem user@host：使用密钥文件 |
    | scp | 远程复制 | - scp file.txt user@host:/path：上传文件
  • scp user@host:/path/file.txt .：下载文件
  • scp -r dir/ user@host:/path：复制整个目录 |

6. 进程管理

命令 功能 示例
ps 查看进程状态 - ps aux：查看所有进程
  • ps -ef | grep nginx：查找nginx进程
  • ps -u username：查看指定用户的进程
  • ps -o pid,ppid,cmd：自定义输出格式
  • ps --forest：显示进程树 |
    | top/htop | 动态进程监控 | - top：实时系统监控（按q退出）
  • top -u username：监控特定用户的进程
  • htop：增强版（需安装）
  • htop -u username：监控特定用户 |
    | kill | 终止进程 | - kill 1234：终止PID=1234的进程
  • kill -9 1234：强制终止
  • kill -l：列出所有信号
  • killall nginx：终止所有nginx进程 |
    | pkill | 按名称终止进程 | - pkill firefox：终止firefox进程
  • pkill -u username：终止指定用户的进程 |
    | nice/renice | 调整进程优先级 | - nice -n 10 command：以较低优先级启动
  • renice +10 -p 1234：调整运行中的进程 |
    | nohup | 后台运行 | - nohup command &：在后台运行命令，即使终端关闭 |
    | bg/fg/jobs | 任务控制 | - command &：在后台启动命令
  • Ctrl+Z：暂停当前任务
  • bg：将暂停的任务放到后台运行
  • fg：将后台任务调到前台
  • jobs：列出所有后台任务 |

7. 系统信息查看

命令 功能 示例
uname 系统信息 - uname -a：所有系统信息
  • uname -r：内核版本
  • uname -m：硬件架构 |
    | free | 内存使用 | - free -h：以人类可读格式显示内存使用情况
  • free -m：以MB为单位
  • free -s 5：每5秒刷新一次 |
    | df | 磁盘空间 | - df -h：显示所有文件系统使用情况
  • df -i：显示inode使用情况
  • df -T：显示文件系统类型 |
    | du | 目录大小 | - du -sh *：显示当前目录下各文件/目录大小
  • du -h --max-depth=1 /path：显示一级子目录大小
  • du -a | sort -nr | head -10：显示最大的10个文件 |
    | lscpu | CPU信息 | - lscpu：显示CPU架构信息 |
    | lsblk | 块设备信息 | - lsblk：列出所有块设备
  • lsblk -f：显示文件系统信息 |
    | dmidecode | 硬件信息 | - dmidecode -t memory：内存硬件信息
  • dmidecode -t system：系统信息
  • dmidecode -t bios：BIOS信息 |
    | uptime | 系统负载 | - uptime：显示运行时间和负载 |
    | who/w | 登录用户 | - who：显示当前登录用户
  • w：显示登录用户及其活动 |
    | last | 登录历史 | - last：显示最近登录记录
  • last username：显示特定用户的登录历史 |

8. 文件权限管理

命令 功能 示例
chmod 修改权限 - chmod 755 file：设置权限为rwxr-xr-x
  • chmod +x script.sh：添加执行权限
  • chmod -R 644 dir/：递归设置目录下所有文件权限
  • chmod u+w,g-x file：用户添加写权限，组删除执行权限 |
    | chown | 修改所有者 | - chown user:group file：修改文件的用户和组
  • chown -R user:group dir/：递归修改目录及内容
  • chown :group file：只修改组 |
    | chgrp | 修改组 | - chgrp group file：修改文件的组
  • chgrp -R group dir/：递归修改目录及内容 |
    | umask | 默认权限 | - umask：显示当前掩码
  • umask 022：设置新建文件的默认权限 |
    | getfacl/setfacl | 访问控制列表 | - getfacl file：显示文件的ACL
  • setfacl -m u:user:rw file：为用户设置权限
  • setfacl -m g:group:rx file：为组设置权限 |

9. 压缩与解压

命令 功能 示例
tar 打包/解包 - tar -cvf archive.tar files/：创建归档
  • tar -xvf archive.tar：解包
  • tar -czvf archive.tar.gz files/：创建gzip压缩包
  • tar -xzvf archive.tar.gz：解压gzip压缩包
  • tar -cjvf archive.tar.bz2 files/：创建bzip2压缩包
  • tar -xjvf archive.tar.bz2：解压bzip2压缩包 |
    | gzip/gunzip | gzip压缩/解压 | - gzip file：压缩文件（替换原文件）
  • gzip -k file：保留原文件
  • gunzip file.gz：解压文件 |
    | zip/unzip | zip压缩/解压 | - zip archive.zip files/：创建zip压缩包
  • zip -r archive.zip dir/：递归压缩目录
  • unzip archive.zip：解压zip文件
  • unzip -l archive.zip：列出zip内容不解压 |

10. 用户与权限管理

命令 功能 示例
useradd 创建用户 - useradd username：创建新用户
  • useradd -m -s /bin/bash username：创建用户并设置shell
  • useradd -G group1,group2 username：创建用户并添加到组 |
    | usermod | 修改用户 | - usermod -aG sudo username：将用户添加到sudo组
  • usermod -s /bin/zsh username：修改用户shell
  • usermod -L username：锁定用户账户 |
    | userdel | 删除用户 | - userdel username：删除用户
  • userdel -r username：删除用户及其主目录 |
    | passwd | 修改密码 | - passwd：修改当前用户密码
  • passwd username：修改指定用户密码 |
    | groupadd | 创建组 | - groupadd groupname：创建新组 |
    | groupmod | 修改组 | - groupmod -n newname oldname：重命名组 |
    | groupdel | 删除组 | - groupdel groupname：删除组 |
    | su | 切换用户 | - su username：切换到指定用户
  • su -：切换到root用户（完整环境） |
    | sudo | 以其他用户执行 | - sudo command：以root权限执行命令
  • sudo -u username command：以指定用户执行命令 |
    | visudo | 编辑sudo配置 | - visudo：安全地编辑/etc/sudoers文件 |
