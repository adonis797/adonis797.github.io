Redis 完全指南：基础命令与Java实战

Redis（Remote Dictionary Server）是一个开源的内存数据结构存储系统，可以用作数据库、缓存和消息中间件。本文将详细介绍Redis的基础概念、常用命令，以及在Java中如何操作Redis。

1. Redis 简介

1.1 什么是 Redis？

Redis 是一个基于内存的键值对数据库，具有以下特点：

  • 高性能：数据存储在内存中，读写速度极快
  • 丰富的数据类型：支持字符串、哈希、列表、集合、有序集合等
  • 持久化：支持数据持久化到磁盘
  • 分布式：支持主从复制和集群模式
  • 事务：支持简单的事务操作

1.2 Redis 的应用场景

  • 缓存系统：提高数据库查询性能
  • 会话存储：分布式系统中的用户会话管理
  • 消息队列：简单的发布订阅和队列功能
  • 计数器：网站访问量、点赞数等统计
  • 排行榜：游戏积分排行、热门文章排序

2. Redis 数据类型与常用命令

2.1 字符串（String）

字符串是Redis最基本的数据类型，一个键最多能存储512MB的数据。

2.2 哈希（Hash）

哈希是一个键值对集合，特别适合存储对象。

2.3 列表（List）

列表是一个有序的字符串列表，支持双向操作。

2.4 集合（Set）

集合是一个无序的字符串集合，元素唯一。

2.5 有序集合（Sorted Set）

有序集合在集合的基础上增加了分数（score），用于排序。
